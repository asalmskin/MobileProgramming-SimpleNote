
SimpleNote Android
Requirements
Android Studio Jellyfish or newer, JDK 17, Gradle wrapper will be generated by Android Studio on first sync.
How to run
1. Open this folder in Android Studio.
2. Wait for Gradle sync. If prompted, let Studio create a Gradle wrapper.
3. Run the app on a device or emulator with internet access.
Configuration
The app targets the backend at https://simple.darkube.app/ by default. If your backend domain differs, update Repository baseUrl in data/repo/Repository.kt.
Features
Registration, login with JWT, token refresh, offline-first notes via Room, search, pagination, CRUD, profile, logout.
